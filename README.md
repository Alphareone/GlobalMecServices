GlobalMec es una landing page de alto impacto visual y rendimiento optimizado, diseñada específicamente para servicios de mecánica de élite. El proyecto ha sido transformado de un diseño básico a una interfaz profesional con enfoque en la conversión de clientes y la confianza visual.

---

# Propuesta de Valor
Esta versión del sitio web no es solo una página informativa, es una herramienta de ventas diseñada bajo los criterios de:
- Profesionalismo: Estética de taller de alta gama.
- Velocidad: Código limpio que carga en milisegundos.
- Confianza: Secciones claras de equipo experto y testimonios de capacidad técnica.

---

# Mejoras Implementadas (Versión 1.1)

# Interfaz de Usuario (UI)
- Glassmorphism: Implementación de tarjetas con efecto de cristal esmerilado para un look moderno.
- Tipografía y Jerarquía: Uso de *Poppins* con pesos visuales calculados para guiar la lectura.
- Paleta Industrial: Uso de colores oscuros (`#020617`) combinados con acentos en naranja vibrante (`#F97316`) para transmitir energía y precisión.

# Rendimiento y UX
- Intersection Observer API: Sustitución de eventos de scroll pesados por una detección inteligente de elementos para animaciones fluidas.
- Mobile First: El diseño ha sido reconstruido para que la experiencia en celulares sea prioritaria y perfecta.
- Iconografía Lucide: Sustitución de imágenes pesadas y emojis por iconos vectoriales ligeros y elegantes.
